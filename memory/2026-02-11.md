## 短视频脚本生成系统 (00:39-00:52)

### 需求演进
1. **初始需求**: 把热点新闻写成15秒小剧本
2. **进阶需求**: 要有"流量嗅觉"，识别真正能火的新闻（如"伊朗总统被逮捕"）

### 系统架构

#### 第一版：基础脚本生成器 ✅
- **文件**: `video-script-generator.py`
- **功能**: 从简报提取top 5，生成标准15秒脚本
- **问题**: 没有筛选机制，可能包含低质量内容

#### 第二版：爆点识别器 ✅
- **文件**: `viral-news-detector.py`
- **核心**: 多维度评分算法
- **功能**: 自动识别高流量潜力新闻

### 爆点识别算法

#### 评分维度（总分可达50+）
1. **关键词权重** (0-10分)
   - 政治军事: 逮捕(10)、总统(9)、战争(9)
   - 科技突破: AGI(10)、突破(8)、颠覆(9)
   - 商业金融: 破产(9)、暴涨(8)、百亿(9)
   - 争议话题: 丑闻(9)、泄露(8)、封杀(8)

2. **情绪词加分** (0-8分)
   - 震惊/炸了/爆了: 8分
   - 突发/紧急: 8-9分

3. **数字冲击力** (每个+5分)
   - "80亿"、"10万人"等大数字

4. **时效性** (+6分)
   - "刚刚"、"突发"、"最新"

5. **冲突性** (+5分)
   - "A vs B"、"对抗"

6. **权威来源** (+10分)
   - Tavily来源（新闻媒体）

7. **Spam惩罚** (-20分/个)
   - "免费信号"、"加入群"

#### 爆点分级
| 分数 | 等级 | 预估流量 | 示例 |
|------|------|---------|------|
| 50+ | 🔥🔥🔥 | 10万+ | 伊朗总统被逮捕 |
| 30-49 | 🔥🔥 | 5万+ | OpenAI CEO辞职 |
| 20-29 | 🔥 | 1万+ | 新AI模型发布 |
| 10-19 | 普通 | 5000+ | 科技公司动态 |

### 爆款脚本特点

#### 结构优化
```
【0-2秒】强力开场
🚨 重大突发！（闪烁特效）

【3-8秒】核心爆点
标题 + 关键信息（放大特写）

【9-12秒】冲击解读
"这件事影响有多大？"（震动特效）

【13-15秒】强力CTA
"关注我，第一时间看热点！"（闪烁）
```

#### 视觉增强
- 震撼开场（闪光/爆炸效果）
- 快速剪辑（0.5秒/镜头）
- 红色/黄色警示色调
- 字幕特大 + 特效（闪烁/震动/放大）

### 当前问题与改进

#### 问题1: 今天新闻分数低（最高11分）
**原因**:
- 大部分是spam（加密货币广告）
- 缺少真正的"大事件"
- Twitter内容质量参差不齐

**改进方向**:
1. 增加新闻源（Reuters、BBC、微博热搜）
2. 优化关键词库（更多中文热点词）
3. 实时监控（每小时检查）

#### 问题2: 需要更敏锐的"新闻嗅觉"
**解决方案**:
- 建立"爆款数据库"（记录哪些新闻真的火了）
- 根据实际播放数据反哺算法
- 学习抖音/快手的热门规律

### 完整工作流

```bash
# 每天执行
1. python3 enhanced-news-aggregator.py  # 聚合新闻
2. python3 viral-news-detector.py       # 识别爆点
3. 使用 viral-scripts 制作视频
4. 发布到抖音/快手（19-22点）
5. 追踪数据，优化算法
```
