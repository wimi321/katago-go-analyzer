### 推荐方案

#### 方案A: OpenCV的findChessboardCorners（最简单）
**优点**:
- 官方实现，稳定可靠
- 自动检测所有内角点（18x18=324个）
- 亚像素精度
- 一行代码搞定

**实现**:
```python
def detect_corners_opencv(self, image):
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    ret, corners = cv2.findChessboardCorners(gray, (18, 18))
    
    if ret:
        # 亚像素优化
        criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 30, 0.001)
        corners = cv2.cornerSubPix(gray, corners, (11, 11), (-1, -1), criteria)
        
        # 提取四个外角点
        outer_corners = [corners[0][0], corners[17][0], corners[-1][0], corners[-18][0]]
        return outer_corners, corners
    
    return None, None
```
